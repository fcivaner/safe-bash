# safe-bash

Command line tools for safer bash usage.

## installation

To install run the following command in your terminal:

```npm install -g safe-bash```

## usage

Confirm:

```text
confirm: utility to confirm commands before execution.

usage: confirm CMD
```

```text
history-clean-pattern: utility to clean command line history of certain records.

usage: history-clean-pattern REGEX_PATTERN
```

```text
edit-execute: utility to edit shell commands before execution.
```
